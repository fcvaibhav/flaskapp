Summary of Forbes 2018 List

Vaibhav Pandey · 2019/05/18 · 2 minute read

Using forbesListR package

We’ll access forbes list through forbesListR package. For more details see https://github.com/abresler/forbesListR.

Let’s begin by loading the packages and calling the Forbes 400 list

forbes400_data <- 
  get_year_forbes_list_data(list = "Forbes 400", year = 2018)

Let’s start by printing top 10 entries in the list

forbes400_data %>% 
  mutate(`net worth (in Bn USD)` = round(net_worth.millions/1000,2),
         image_url = paste0("![](",url.image, ")")) %>% 
  select(image_url, name, `net worth (in Bn USD)`, age, industry, headquarters) %>%
  head(10) %>%
  kable(caption = "Top 10 in Forbes 2018") %>%
  kable_styling(bootstrap_options = c("striped", "hover"))
