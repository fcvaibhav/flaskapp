Forbes 2019 List

Vaibhav Pandey · 2019/05/18 · 2 minute read

using forbesListR package

We’ll access forbes list through forbesListR package. For more details see https://github.com/abresler/forbesListR.

Let’s begin by calling the Forbes 400 list through

library(forbesListR)
library(dplyr)
forbes400_values <- 
  get_year_forbes_list_data(list = "Forbes 400", year = 2018)

forbes400_values %>% 
  mutate(`full name` = paste(name, last_name)) %>% 
  rename(`net worth (in million USD)` = net_worth.millions) %>% 
  select(`full name`, `net worth (in million USD)`, age, industry, headquarters)
